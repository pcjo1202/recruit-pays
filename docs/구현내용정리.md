# 구현 내용

## 요구사항 정의
> "결제/가맹점 관련 데이터를 시각화하는 대시보드 화면 페이지를 구현"
> 주어진 API를 통해 결제대행사(PG) 도메인에 적합한 내용을 구성

### 주요 기능
1. 첫 대시보드 페이지 (/dashboard)
> 결제/가맹점 관련 데이터를 시각화하는 대시보드 화면 페이지
- [x] 총 거래액 표시
- [x] 총 거래 건수
- [x] 전체 가맹점 수
- [x] 활성 가맹점 수
- [ ] 결제 상태별 분포 (도넛 차트)
- [ ] 결제 수단별 통계 (바 차트)
- [ ] 가맹점 상태 분포 (바 차트)
- [ ] 업종별 가맹점 분포 (파이 차트)

2. 거래 내역 리스트 페이지 (/payments)
> 전체 거래 내역을 조회할 수 있는 리스트 페이지
- [x] 기간 필터링
- [x] 결제 상태 필터링
- [x] 결제 수단 필터링
- [ ] 가맹점 검색 (가맹점명으로 검색)
- [ ] 정렬 (날짜, 금액, 상태)
- [x] 페이지네이션 (10건씩 표시)
- [x] 필터된 결과의 요약 (총 거래액, 건수 표시)

3. 가맹점 페이지 (/merchants)
> 전체 가맹점을 조회할 수 있는 리스트 페이지 
- [ ] 가맹점 상태 필터링 - (status) 기준
- [ ] 업종 필터 - (bizType) 기준
- [ ] 가맹점 검색 (가맹점명으로 검색)
- [ ] 정렬 (거래액, 거래건수, 가맹점명, 등록일 순)
- [ ] 페이지네이션 (10건씩 표시)
- [ ] 필터된 결과의 요약 (총 가맹점 수 표시)

4. 가맹점 상세 페이지 (/merchants/{mchtCode})
> 가맹점 상세 정보를 확인할 수 있는 페이지
- [ ] 총 거래액
- [ ] 총 거래 건수
- [ ] 평균 거래액
- [ ] 승인율
- [ ] 일별 거래 통계 (바 차트)
- [ ] 결제 수단별 분포 (파이 차트)
- [ ] 거래 상태별 분포 (바 차트)

## 주요 API 
### 거래 내역
- GET /api/v1/payments/list - 거래 내역 전체 조회

```json
{
  "status": 0,
  "message": "string",
  "data": [
    {
      "paymentCode": "string",
      "mchtCode": "string",
      "amount": "string",
      "currency": "string",
      "payType": "ONLINE",
      "status": "PENDING",
      "paymentAt": "2025-11-18T07:26:29.898Z"
    }
  ]
}

```

### 가맹점 정보
- GET /api/v1/merchants/list - 가맹점 목록 조회

```json
{
  "status": 0,
  "message": "string",
  "data": [
    {
      "mchtCode": "string",
      "mchtName": "string",
      "status": "string",
      "bizType": "string"
    }
  ]
}

```

- GET /api/v1/merchants/details - 가맹점 상세 조회

```json
{
  "status": 0,
  "message": "string",
  "data": [
    {
      "mchtCode": "string",
      "mchtName": "string",
      "status": "string",
      "bizType": "string",
      "bizNo": "string",
      "address": "string",
      "phone": "string",
      "email": "string",
      "registeredAt": "2025-11-18T07:27:01.038Z",
      "updatedAt": "2025-11-18T07:27:01.038Z"
    }
  ]
}

```


- GET /api/v1/merchants/details/{mchtCode} - 가맹점 코드로 상세 조회

```json
{
  "status": 0,
  "message": "string",
  "data": {
    "mchtCode": "string",
    "mchtName": "string",
    "status": "string",
    "bizType": "string",
    "bizNo": "string",
    "address": "string",
    "phone": "string",
    "email": "string",
    "registeredAt": "2025-11-18T07:27:09.829Z",
    "updatedAt": "2025-11-18T07:27:09.829Z"
  }
}

```
