# 채용 과제

## 개발 환경

- Node.js: 20.19.5
- npm
- TypeScript 5
- Next.js 16.0.3, React 19.2.0
- Tailwind CSS 4
- Shadcn UI

## 환경 변수

```
NEXT_PUBLIC_API_BASE_URL=https://recruit.paysbypays.com/api/v1
```

## 실행

```bash
# 의존성 설치
npm install

# 실행
npm run dev
```
